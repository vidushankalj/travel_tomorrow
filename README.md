# Travel Tomorrow 🚀

A modern travel planning web application to help you discover, plan, and dream about your next adventure — **tomorrow could be the start of your journey**.

Built with **React**, **Vite**, **Tailwind CSS**, and a separate backend service.

## ✨ Features

- 🌍 Discover destinations with beautiful visuals and details
- 🗺️ Build personalized travel itineraries
- 🔍 Smart search & filters for flights, hotels, activities
- 📱 Fully responsive design (mobile-first with Tailwind)
- ⚡ Lightning-fast development & builds using Vite
- 🛠️ Separate backend integration (Node.js / Express / other – add your stack here)

## 🚀 Quick Start

### Prerequisites
- Node.js ≥ 18
- npm or pnpm / yarn

### Development

```bash
# Clone the repo
git clone https://github.com/vidushankalj/travel_tomorrow.git
cd travel_tomorrow

# Install dependencies
npm install
# or
pnpm install
# or
yarn install

# Start frontend dev server<a href="http://localhost:5173" target="_blank" rel="noopener noreferrer nofollow"></a>
npm run dev
